\doxysubsection{Detailed Description}
Allows user to create and manage windows. 

\hypertarget{class_boreal_1_1_window_Description}{}\doxysubsection{Description}\label{class_boreal_1_1_window_Description}
The window class is used to create and manage windows. It allows the user to do the following\+:
\begin{DoxyItemize}
\item Create window with parameters
\item Change window width \& height
\item Change window title
\item Update window for main loop 
\end{DoxyItemize}
