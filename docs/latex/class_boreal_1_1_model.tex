\doxysubsection{Detailed Description}
Allows user to create and manage models. 

\hypertarget{class_boreal_1_1_window_Description}{}\doxysubsection{Description}\label{class_boreal_1_1_window_Description}
The model class is used to create and manage models. It allows the user to do the following\+:
\begin{DoxyItemize}
\item Create model with vertices, indexes, shaders, textures
\item Change location, rotation, and scale of the model
\item Move and rotate model 
\end{DoxyItemize}
